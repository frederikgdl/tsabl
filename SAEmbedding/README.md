# SAEmbedding - Tang 2016
Implementation of Tang 2016 in Java

## Setup
1. Make sure you have java, javac and ant installed.
2. Give the run script permission using:
```bash
chmod u+x run.sh
```

## Build
To build enter:
```bash
ant
```

## Run
Change parameters in run.sh and then enter:
```bash
./run.sh
```
